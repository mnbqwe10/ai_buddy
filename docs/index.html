What it does

A compact toolbar for repeated AI tasks

AI Buddy keeps prompt actions close to the text you are reading, while leaving configuration and platform behavior under your control.

Selection toolbar

Highlight text and launch common actions directly from the page.

Reusable scenarios

Group actions for learning, workplace, creative, or your own workflows.

Side panel chat

Keep the source page visible while the chat platform handles the prompt.

Prompt control

Customize action templates, colors, toolbar order, and response language.

Platform-aware sending

AI chat platforms can auto-send; messaging platforms draft until you allow it.

Local configuration

The extension stores preferences and actions, not your prompts or conversations.

Screenshots

Built around the browser surfaces you already use

Local install

Load AI Buddy as an unpacked extension

Build the repository locally, then load the generated dist/ folder from Chrome's extensions page.

  1. npm install
  2. npm run build
  3. Open chrome://extensions and enable Developer mode.
  4. Choose Load unpacked and select dist/.
  5. Pin AI Buddy and select your scenario, platform, and language.

Personal project, privacy-minded behavior

AI Buddy is not affiliated with any AI chat or messaging platform. You sign in to your own platform account, and selected text is sent only when you trigger an action.
