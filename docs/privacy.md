# Privacy Policy for AI Buddy

Last updated: May 2026

AI Buddy is an open-source Chrome extension that provides user-initiated AI actions for selected webpage text.

## Data Processing

AI Buddy may process:
- user-selected webpage text,
- AI prompts,
- webpage content required for extension functionality,
- user settings and preferences.

Processing occurs only when explicitly initiated by the user.

## Third-Party Services

When the user requests AI functionality, selected content may be securely transmitted over HTTPS to supported third-party AI services, including:
- OpenAI ChatGPT
- Anthropic Claude
- Google Gemini
- Microsoft Copilot
- DeepSeek

Data is processed solely to generate the requested AI response.

## Data Collection

AI Buddy does not:
- sell user data,
- use advertising trackers,
- collect browsing history,
- perform hidden background monitoring,
- collect passwords or authentication credentials.

## Local Storage

Extension settings and preferences are stored locally using Chrome extension storage APIs.
