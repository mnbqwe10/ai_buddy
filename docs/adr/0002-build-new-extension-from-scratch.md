# Build the new extension from scratch

The repository contains a working prototype under `code_snippets/chatgpt_sidebar`, but the new product introduces **Scenarios**, an **Action Library**, platform send policies, richer Options management, and messaging-platform safety rules that do not fit the prototype's fixed action arrays. We decided to build the new extension architecture from scratch and use the prototype only as a behavior reference for selection toolbar interactions, side panel prompt delivery, platform bridges, and starter prompt ideas.
